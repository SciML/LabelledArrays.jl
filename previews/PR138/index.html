LabelledArrays.jl: Arrays with Label Goodness

LabelledArrays.jl is a package which provides arrays with labels, i.e. they are arrays which map, broadcast, and all of that good stuff, but their components are labelled. For instance, users can name the second component of an array to :second and retrieve it with A.second.

Installation

To install LabelledArrays.jl, use the Julia package manager:

using Pkg
Pkg.add("LabelledArrays")
